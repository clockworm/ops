LarryMS 是完全基于经典模块化前端框架Layui的一款 后台管理系统

运行需要放置在localhost环境下；建议源码解压放根目录

目录说明：
backstage/  后台html css文件
common/  layui框架 及larryMS扩展核心文件
   common/lib 后台js及扩展组件
   common/js  智能模块加载
   common/plus 第三方jq插件目录
   common/layui 可同步layui官方最新版本
